# flask-contacts-app
My firs flask app

## Set VENV OS Python3
1. mkdir myproject<br>
2. cd myproject<br>
3. python3 -m venv venv <br>
4. venv\Scripts\activate 

## Install flask on VENV
pip install Flask<br>
pip install flask-mysqldb

### Isues
sudo apt-get install libmysqlclient-dev

## Running the App (Linux)
python3 App.py
